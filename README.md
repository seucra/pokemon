# 🎮 PokemonCraze

A high-performance, premium Pokémon data hub and team builder designed for enthusiasts. Built with a modern tech stack to provide a sleek, "nerd-first" experience.

## ✨ Features

- **🔍 Intelligent Search**: Instant Pokémon lookup using PokéAPI with smart autocomplete.
- **🛡️ Team Builder**: Manage a full team of 6 Pokémon with interactive slot selection.
- **📊 Nerd-Hub Statistics**: Detailed base stat visualizations and ability analysis.
- **⚔️ Defensive Coverage**: Real-time calculation of type weaknesses (2x/4x), resistances, and immunities.
- **⚡ Performance First**: Built with Vite 7 and Tailwind CSS v4 for lightning-fast interactions and tiny bundle sizes.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite 7](https://vite.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **API**: [PokéAPI](https://pokeapi.co/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
